Cours : Les tableaux

Introduction

Les tableaux sont exclusivement réservés à la présentation de données.
Il n’est plus question dans une application Web moderne d’utiliser les tableaux pour réaliser la disposition de la page.
Nous vous les montrons car ils sont encore utilisés pour la génération de template d'email.
